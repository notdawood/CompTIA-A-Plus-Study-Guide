# 💻 Network Commands

## 📌 ipconfig
Displays IP configuration of the device

### Usage:
ipconfig

### Important Options:
- ipconfig /all → Detailed info
- ipconfig /release → Release IP
- ipconfig /renew → Request new IP

---

## 📌 ping
Tests connectivity between devices

### Usage:
ping 8.8.8.8

### Notes:
- Replies = Connection is working
- Request timed out = Issue exists

---

## 📌 tracert
Shows the path packets take to a destination

### Usage:
tracert google.com

---

## 📌 nslookup
Used to query DNS servers

### Usage:
nslookup google.com

---

## 📌 netstat
Displays network connections and ports

### Usage:
netstat -an
