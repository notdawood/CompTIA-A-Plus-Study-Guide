# 🖥️ Introduction to IT & Hardware

## 📌 What is a Computer?
A computer is an electronic device that processes data into useful information.

## ⚙️ Basic Components of a Computer
- Input Devices (Keyboard, Mouse)
- Processing (CPU)
- Output Devices (Monitor, Printer)
- Storage Devices (HDD, SSD)

---

## 🧠 What is Hardware?
Hardware refers to all the physical components of a computer that you can touch.

### Examples:
- CPU
- RAM
- Hard Drive
- Motherboard
- Power Supply

---

## 💻 What is Software?
Software is the set of instructions that tells the hardware what to do.

### Types:
- Operating System (Windows, Linux)
- Applications (Browser, Word)

---

## 🔄 Input → Process → Output Cycle
1. Input: User enters data
2. Process: CPU processes it
3. Output: Result is displayed

---

## 🔥 Important Notes (Exam Focus)
- Hardware = Physical
- Software = Programs
- CPU is the brain of the computer
