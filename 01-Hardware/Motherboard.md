# 🧩 Motherboard

## 📌 Definition
The motherboard is the main circuit board that connects all computer components together.

## 🧠 Key Components
- CPU Socket
- RAM Slots
- Chipset
- Power Connectors
- Expansion Slots (PCIe)

## ⚙️ Functions
- Connects all hardware components
- Allows communication between devices
- Distributes power

## 🔥 Important Notes (Exam Focus)
- Different motherboards support different CPUs
- Form factors: ATX, Micro-ATX, Mini-ITX

## 🛠️ Troubleshooting
- PC not turning on → Check power connectors
- No display → Check GPU / RAM / CPU seating
