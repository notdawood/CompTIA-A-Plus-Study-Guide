# 🧠 CPU (Central Processing Unit)

## 📌 Definition
The CPU is the brain of the computer that processes instructions and performs calculations.

## 🧠 Key Concepts
- Cores: Number of processing units
- Threads: Tasks handled simultaneously
- Clock Speed (GHz): Processing speed

## ⚙️ Types
- Intel
- AMD

## 🔥 Important Notes (Exam Focus)
- More cores = better multitasking
- Higher clock speed = faster performance

## 🛠️ Troubleshooting
- Overheating → Check cooling system
- Slow performance → Check CPU usage
