# 💾 Storage Devices

## 📌 Definition
Storage devices are used to store data permanently.

## ⚙️ Types
- HDD (Hard Disk Drive)
- SSD (Solid State Drive)

## 🧠 Differences
- HDD: Slower, cheaper, mechanical
- SSD: Faster, more expensive, no moving parts

## 🔥 Important Notes (Exam Focus)
- SSD improves system performance
- Storage is non-volatile

## 🛠️ Troubleshooting
- Slow system → Upgrade to SSD
- Disk not detected → Check connections
