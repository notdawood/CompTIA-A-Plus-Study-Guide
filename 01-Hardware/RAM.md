# 🧠 RAM (Random Access Memory)

## 📌 Definition
RAM is temporary memory used to store data that the CPU is actively using.

## 🧠 Key Concepts
- Volatile memory (data lost when power is off)
- Measured in GB

## ⚙️ Types
- DDR4
- DDR5

## 🔥 Important Notes (Exam Focus)
- More RAM = better multitasking
- Not a storage device

## 🛠️ Troubleshooting
- Blue screen → Possible RAM issue
- PC not booting → Check RAM installation
