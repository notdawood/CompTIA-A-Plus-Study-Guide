# 🧠 BIOS / UEFI

## 📌 Definition
BIOS (Basic Input Output System) and UEFI (Unified Extensible Firmware Interface) are firmware interfaces that start the computer and initialize hardware.

## 🧠 Functions
- Performs POST (Power-On Self Test)
- Initializes hardware components
- Loads the operating system

## ⚙️ Differences

- BIOS: Older, slower interface
- UEFI: Modern, faster, supports GUI and larger drives

## 🔥 Important Notes (Exam Focus)
- UEFI replaced BIOS in modern systems
- Accessed during boot (e.g., F2, DEL keys)

## 🛠️ Troubleshooting
- Boot issues → Check boot order in BIOS/UEFI
- Wrong boot device → Adjust boot priority
