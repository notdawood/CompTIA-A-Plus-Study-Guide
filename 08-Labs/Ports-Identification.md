# 🧪 Lab: Identify Common Ports

## 🎯 Objective
Identify different ports on a computer or laptop.

## 🛠️ Task
Find or observe a device and identify the following ports:

- USB
- HDMI
- Ethernet
- Audio Jack

## 🧠 What You Need to Do
Write the function of each port in one line.

## 💡 Example:
USB → Used to connect peripherals like keyboard, mouse, and flash drives
