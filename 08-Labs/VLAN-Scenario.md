# 🧪 Lab: VLAN Scenario

## 🎯 Objective
Understand how VLANs are used in real environments.

## 🛠️ Scenario
A company has 3 departments:
- HR
- IT
- Sales

## 🧠 Task
Explain how you would:

- Assign each department to a VLAN
- Improve security using VLANs

## 💡 Example:
HR → VLAN 10  
IT → VLAN 20  
Sales → VLAN 30  

Each department is isolated for security purposes.
