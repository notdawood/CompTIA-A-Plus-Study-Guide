# 🧪 Lab: Identify Computer Hardware

## 🎯 Objective
To identify and understand the basic hardware components of a computer.

## 🛠️ Task
Open your computer case (if possible) or search online images and identify the following components:

- CPU
- RAM
- Hard Drive (HDD/SSD)
- Power Supply

## 🧠 What You Need to Do
Write one line describing the function of each component.

## 💡 Example:
CPU → Processes instructions and performs calculations
RAM → Temporarily stores data for quick access
