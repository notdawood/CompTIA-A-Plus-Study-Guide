# 🧪 Lab: Identify IP Addresses

## 🎯 Objective
Understand how to identify and classify IP addresses.

## 🛠️ Task
Find or list 3 IP addresses and determine:

- Whether they are private or public
- Whether they are IPv4 or IPv6

## 🧠 What You Need to Do
Explain your classification for each IP.

## 💡 Example:
192.168.1.10 → Private IPv4 address
8.8.8.8 → Public IPv4 address
