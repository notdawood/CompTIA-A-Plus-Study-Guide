# 🧪 Lab: Network Commands Practice

## 🎯 Objective
Practice basic network troubleshooting using commands

## 🛠️ Task

Run the following commands on your computer:

- ipconfig
- ping 8.8.8.8
- tracert google.com
- nslookup google.com

## 🧠 What You Need to Do
Write what each command shows and what you learned from it.

## 💡 Example:
ping 8.8.8.8 → Shows if internet connectivity is working
