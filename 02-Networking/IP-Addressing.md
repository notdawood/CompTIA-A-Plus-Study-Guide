# 🌐 IP Addressing

## 📌 Definition
An IP address is a unique identifier assigned to each device on a network.

## 🧠 Types of IP Addresses

### IPv4
- Format: 192.168.1.1
- 32-bit address
- Most commonly used

### IPv6
- Longer format
- Designed to replace IPv4
- Example: 2001:0db8::1

## ⚙️ Types of IP
- Static IP (manually assigned)
- Dynamic IP (assigned by DHCP)

## 🔥 Important Notes (Exam Focus)
- Private IP ranges:
  - 192.168.x.x
  - 10.x.x.x
  - 172.16.x.x – 172.31.x.x

## 🛠️ Troubleshooting
- No network → Check IP configuration
- IP conflict → Two devices using same IP
