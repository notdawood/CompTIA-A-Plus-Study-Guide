# 🌐 Network Devices

## 📌 Router
- Connects different networks together
- Routes traffic between networks
- Assigns IP addresses (via DHCP in many cases)

## 📌 Switch
- Connects devices within the same network (LAN)
- Uses MAC addresses to forward data
- Operates at Layer 2 (Data Link Layer)

## 📌 Access Point (AP)
- Provides wireless connectivity (Wi-Fi)
- Connects wireless devices to a wired network

## 📌 Modem
- Connects a home network to the internet
- Converts signals from ISP to usable data

## 🔥 Important Notes (Exam Focus)
- Router → Network to Network
- Switch → Device to Device (LAN)
- AP → Wireless connectivity
