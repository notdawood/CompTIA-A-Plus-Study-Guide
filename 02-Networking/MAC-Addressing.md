# 🔗 MAC Address

## 📌 Definition
A MAC address is a unique hardware identifier assigned to a network interface card (NIC).

## 🧠 Format
- Example: 00:1A:2B:3C:4D:5E
- 48-bit address
- Assigned by manufacturer

## ⚙️ Key Points
- Works at Data Link Layer
- Used for local network communication

## 🔥 Important Notes (Exam Focus)
- MAC address is permanent (burned into hardware)
- IP address can change, MAC address usually does not

## 🛠️ Troubleshooting
- Network filtering → MAC filtering may block devices
- Duplicate MAC (rare) → Network issues
