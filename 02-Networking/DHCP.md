# 📡 DHCP (Dynamic Host Configuration Protocol)

## 📌 Definition
DHCP is a protocol that automatically assigns IP addresses to devices on a network.

## 🧠 How It Works (DORA Process)

1. Discover → Device looks for DHCP server
2. Offer → Server offers an IP address
3. Request → Device requests the offered IP
4. Acknowledge → Server confirms assignment

## ⚙️ Key Points
- Saves time (no manual IP configuration)
- Prevents IP conflicts

## 🔥 Important Notes (Exam Focus)
- DHCP assigns:
  - IP Address
  - Subnet Mask
  - Default Gateway
  - DNS Server

## 🛠️ Troubleshooting
- No IP → Check DHCP server
- APIPA (169.254.x.x) → DHCP failure
