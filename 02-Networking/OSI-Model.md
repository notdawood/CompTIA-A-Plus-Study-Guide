# 📶 OSI Model

## 📌 Definition
The OSI Model is a conceptual framework used to understand how data travels across a network.

## 🧠 The 7 Layers

1. Physical Layer
2. Data Link Layer
3. Network Layer
4. Transport Layer
5. Session Layer
6. Presentation Layer
7. Application Layer

## ⚙️ Functions

- Physical → Cables, signals
- Data Link → MAC addresses
- Network → IP addressing & routing
- Transport → TCP/UDP
- Session → Connection management
- Presentation → Data formatting/encryption
- Application → User interaction (HTTP, FTP, etc.)

## 🔥 Important Notes (Exam Focus)
- Each layer has a specific role
- Troubleshooting often starts from Physical layer
