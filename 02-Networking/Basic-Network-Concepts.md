# 🌐 Basic Networking Concepts

## 📌 Bandwidth
- Maximum data transfer rate of a network
- Measured in Mbps or Gbps

## 📌 Latency
- Delay in data transmission
- Lower latency = better performance

## 📌 Throughput
- Actual data transfer rate achieved

## 📌 Packet
- Small unit of data transmitted over a network

## 📌 Protocol
- Rules that define communication between devices
- Examples: HTTP, FTP, TCP, IP

## 🔥 Important Notes (Exam Focus)
- Bandwidth ≠ Throughput
- Latency affects performance
- Packets are the building blocks of data transfer
