# 🌍 DNS (Domain Name System)

## 📌 Definition
DNS translates domain names into IP addresses.

## 🧠 Example
google.com → 142.250.190.78

## ⚙️ How It Works
- User enters domain name
- DNS server resolves it to IP
- Device connects to the server

## 🔥 Important Notes (Exam Focus)
- Humans use names (google.com)
- Computers use IP addresses

## 🛠️ Troubleshooting
- Website not loading → Check DNS
- Use ping or nslookup to test DNS
