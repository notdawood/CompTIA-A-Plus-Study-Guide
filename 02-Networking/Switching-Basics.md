# 🔀 Switching Basics

## 📌 Definition
A switch is a network device that connects multiple devices within the same Local Area Network (LAN).

## 🧠 How It Works
- Uses MAC addresses to forward data
- Learns devices by building a MAC address table
- Sends data only to the intended device (unlike hubs)

## ⚙️ Key Concepts

### MAC Address Table
- Stores MAC addresses and their corresponding ports
- Helps the switch send data efficiently

### Unicast
- Data sent to a single device

### Broadcast
- Data sent to all devices in the network

### Collision Domain
- Each switch port is its own collision domain

## 🔥 Important Notes (Exam Focus)
- Switch operates at Layer 2 (Data Link Layer)
- More efficient than hubs
- Reduces network congestion

## 🛠️ Troubleshooting
- No connectivity → Check cable and port status
- Wrong device communication → Check MAC table
