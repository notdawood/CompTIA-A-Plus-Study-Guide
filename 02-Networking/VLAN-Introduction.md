# 🧩 VLAN (Virtual LAN)

## 📌 Definition
A VLAN is a logical grouping of devices within a network, even if they are not physically connected to the same switch.

## 🧠 Why VLANs?

- Improve security
- Reduce broadcast traffic
- Organize network logically (e.g., HR, IT, Sales)

## ⚙️ Key Concepts

### Default VLAN
- All devices are in VLAN 1 by default

### VLAN ID
- Each VLAN has a unique ID (e.g., VLAN 10, VLAN 20)

### Access Port
- Carries traffic for a single VLAN

### Trunk Port
- Carries traffic for multiple VLANs between switches

## 🔥 Important Notes (Exam Focus)
- VLANs separate networks logically
- Devices in different VLANs cannot communicate without a router

## 🛠️ Troubleshooting
- No communication → Check VLAN assignment
- Devices isolated → Check VLAN configuration
