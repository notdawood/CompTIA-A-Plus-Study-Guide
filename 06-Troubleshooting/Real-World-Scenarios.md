# 🛠️ Real-World Troubleshooting Scenarios

## 🎯 Objective
Apply troubleshooting skills to real-world IT problems.

---

## 🧩 Scenario 1: No Internet Connection

### Problem:
User cannot access the internet

### Steps:
1. Check cables / Wi-Fi connection
2. Run ipconfig
3. Check IP address

### Possible Causes:
- DHCP issue
- Cable unplugged
- Router problem

---

## 🧩 Scenario 2: Computer Not Turning On

### Problem:
PC does not power on

### Steps:
1. Check power cable
2. Check PSU
3. Test power button

### Possible Causes:
- Faulty PSU
- Loose cable
- Dead motherboard

---

## 🧩 Scenario 3: Slow Computer

### Problem:
System is very slow

### Steps:
1. Open Task Manager
2. Check CPU and RAM usage
3. Check startup programs

### Possible Causes:
- Too many programs
- Malware
- Low RAM

---

## 🧩 Scenario 4: No Display

### Problem:
Monitor shows no signal

### Steps:
1. Check HDMI/Display cable
2. Check GPU
3. Test monitor

### Possible Causes:
- Loose cable
- GPU issue
- Monitor problem

---

## 🧠 Troubleshooting Methodology

1. Identify the problem
2. Establish a theory
3. Test the theory
4. Implement solution
5. Verify functionality
6. Document everything
