# 🧠 Common Concepts

## 📌 IP vs MAC
- IP → Logical (can change)
- MAC → Physical (fixed)

---

## 📌 TCP vs UDP
- TCP → Reliable
- UDP → Faster

---

## 📌 HDD vs SSD
- HDD → Slow, mechanical
- SSD → Fast, no moving parts

---

## 📌 BIOS vs UEFI
- BIOS → Older
- UEFI → Faster, modern

---

## 📌 Authentication vs Authorization
- Authentication → Who you are
- Authorization → What you can access
