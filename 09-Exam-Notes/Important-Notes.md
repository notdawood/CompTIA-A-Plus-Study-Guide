# 🔥 Important Exam Notes

## ⚡ Networking
- Private IP ranges:
  - 192.168.x.x
  - 10.x.x.x
  - 172.16.x.x – 172.31.x.x

---

## ⚡ Subnetting
- /24 = 255.255.255.0
- First IP → Network
- Last IP → Broadcast

---

## ⚡ DHCP
- Assigns IP automatically

---

## ⚡ DNS
- Converts domain → IP

---

## ⚡ Troubleshooting
Always follow steps:
1. Identify problem
2. Test
3. Fix
4. Verify

---

## ⚡ Windows
- NTFS is default file system
- FAT32 has 4GB file limit
