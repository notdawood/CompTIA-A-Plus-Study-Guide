# 🖥️ Operating System Basics

## 📌 Definition
An Operating System (OS) is software that manages computer hardware and provides services for applications.

## 🧠 Key Functions
- Process management
- Memory management
- File system management
- Device management

## ⚙️ Examples
- Windows
- Linux
- macOS

## 🔥 Important Notes (Exam Focus)
- OS acts as a bridge between user and hardware
- Without OS, computer is unusable
