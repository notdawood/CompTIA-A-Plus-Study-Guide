# 👤 Users and Permissions

## 📌 User Accounts Types

### Administrator
- Full control over system

### Standard User
- Limited access

---

## ⚙️ Permissions

### Read
- View files

### Write
- Modify files

### Execute
- Run programs

### Full Control
- All permissions

---

## 🧠 Key Concepts

- Least Privilege Principle → Give minimum access needed
- Permissions can be assigned to users or groups

---

## 🔥 Important Notes (Exam Focus)
- Administrator has full access
- Permissions control security

## 🛠️ Troubleshooting
- Access denied → Check permissions
- Cannot install software → Need admin rights
