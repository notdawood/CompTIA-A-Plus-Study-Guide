# 📁 File Systems

## 📌 Definition
A file system controls how data is stored and retrieved on storage devices.

## ⚙️ Common Types

- NTFS (Windows default)
- FAT32
- exFAT

## 🧠 Differences

### NTFS
- Supports large files
- Security features (permissions)

### FAT32
- Compatible with many devices
- Limited file size (4GB max)

### exFAT
- Designed for flash drives
- No file size limit like FAT32

## 🔥 Important Notes (Exam Focus)
- NTFS is most commonly used in Windows
- FAT32 has limitations

## 🛠️ Troubleshooting
- File not copying → Check file system limitations
- USB not working → Try different format
