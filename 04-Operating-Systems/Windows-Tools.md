# 🛠️ Windows Administrative Tools

## 📌 Task Manager
- Monitor system performance
- End tasks
- Check CPU, RAM usage

---

## 📌 Device Manager
- View and manage hardware devices
- Update drivers
- Troubleshoot hardware issues

---

## 📌 Disk Management
- Manage partitions
- Format drives
- Assign drive letters

---

## 📌 Event Viewer
- View system logs
- Diagnose errors and warnings

---

## 📌 Services
- Manage background services
- Start/Stop services

---

## 🔥 Important Notes (Exam Focus)
- Event Viewer is key for troubleshooting
- Disk Management is used for storage configuration
