# ⚙️ System Utilities

## 📌 Disk Cleanup
- Removes temporary files
- Frees up disk space

---

## 📌 Disk Defragmentation
- Improves HDD performance
- Rearranges fragmented data

---

## 📌 System Restore
- Restores system to previous state

---

## 📌 Windows Update
- Updates system and security patches

---

## 🔥 Important Notes (Exam Focus)
- Defrag only for HDD (not SSD)
- System Restore helps fix system issues
