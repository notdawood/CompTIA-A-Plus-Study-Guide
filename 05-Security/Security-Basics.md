# 🔐 Security Basics

## 📌 Definition
Security refers to protecting systems, networks, and data from unauthorized access or attacks.

## 🧠 Key Concepts

### Confidentiality
- Only authorized users can access data

### Integrity
- Data should not be altered

### Availability
- Systems should be accessible when needed

---

## ⚙️ Authentication vs Authorization

- Authentication → Who are you?
- Authorization → What can you access?

---

## 🔥 Important Notes (Exam Focus)
- CIA Triad = Confidentiality, Integrity, Availability
- Strong passwords are essential

## 🛠️ Examples
- Login with username/password → Authentication
- Access to files → Authorization
